Программа EditingText производит обработку текста:
•	Открытие текстового файла
•	Удаление специальных символов в тексте, указанных пользователем
•	Удаление слов, содержащих меньше или равно символов в слове в тексте
•	Сохранение текстового файла в указанную папку с именем или изменение текущего редактируемого файла
Для запуска программы необходимо выполнить файл EditingText.exe находящемся в папке «Программа exe»
 
Пользователь выбирает файл нажатием кнопки Открыть файл
 
 
Пользователь может ввести символ который хочет удалить в поле Удалить символ. Нажимает кнопку Выполнить удаление.
 
Пользователь может удалить слова выбранной длины и менее этой длины вводом числа в поле Удалить слова заданной длины. Пользователь нажимает кнопку Удалить слова.
 
Сохранить файл пользователь может 2 способами. Перезаписав текущий файл, для этого необходимо поставить галочку Перезаписать файл. Сохранить в другом месте, необходимо ввести название нового файла в поле Новое название файла. 
Сохранение происходит после нажатия клавиши Сохранить файл
 
Производится сообщение о успешном или нет сохранении.
